# AURA – KI-native Web-Framework

**AURA** ist das erste Web-Framework, das speziell dafür entwickelt wurde, dass **KIs optimal Apps erstellen können**, während Menschen die Kontrolle und Übersicht behalten.

## Inhalt dieser Dokumentation

- [Bedienung für Menschen](human-guide.md)  
- [Referenz für KIs](ai-guide.md)  
- [Komponentenübersicht](components.md)  

AURA basiert auf:

- **Zero-Build**: läuft direkt im Browser (HTML/CSS/JS)  
- **Declarative Intent Schema**: JSON-getrieben  
- **GPU-first Rendering**: Smooth Motion, Chrome-optimiert  
- **Komponentenbasiert**: VideoDeck, VideoGrid etc.
